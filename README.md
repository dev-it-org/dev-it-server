## Installation

```bash

 # Create .env and set your ACCESS and REFRESH TOKENS like in .env.example
 $ yarn
 $ docker-compose up
 $ yarn prisma migrate dev --create-only
 $ yarn prisma db push
 $ yarn prisma studio
 
```

## Running the app

```bash

# Main mode
$ yarn start:dev


# Open Swagger by default: localhost:4000/api-docs

```
